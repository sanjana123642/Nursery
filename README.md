# Flower Nursery Website
This project is a simple and visually pleasant flower nursery website built using basic HTML and CSS. It displays different flowers with images and prices in a clean card-based layout, suitable for learning and practice purposes.

Overview

The website features a header section, a product display area showcasing flowers, and a footer. The design focuses on clarity, soft colors, and easy readability, making it ideal for beginners exploring frontend development.

Features

Clean header with nursery title and tagline

Card-based flower display layout

Images with rounded corners

Soft green color theme inspired by nature

Simple and readable structure

Footer with copyright information

Technologies Used

HTML5

CSS3

Project Structure
index.html

All content, styles, and layout are contained in a single file.

How to Run

Download or copy the project file.

Open index.html in any modern web browser.

The website will load instantly.

No setup or dependencies are required.

Learning Objectives

Practice HTML page structure

Understand basic CSS styling

Learn Flexbox for layout alignment

Create simple card-based UI designs

Limitations

Static content only

No cart or purchase functionality

No responsiveness for very small screens

Possible Enhancements
